EUI-64

EUI-64 (Extended Unique Identifier) is a method we can use to automatically configure IPv6 host addresses (Interface ID). An IPv6 device will use the MAC address of its interface to generate a unique 64-bit interface ID.

I used HTML, CSS and Vanilla JavaScript to make this project. I also wrote this project in python which you can find the repo in my github account.
